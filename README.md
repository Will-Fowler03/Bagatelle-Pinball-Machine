# Bagatelle Pinball Machine

The word **"Bagatelle”** refers to the mechanical nature of this project. This pinball machine was designed, prototyped, and constructed using only mechanical components, with no electronic actuation or control systems.

## The Assignment

- To create an enclosed, fully functional mechanical pinball machine of given dimensional constraints
- To add obstacles of varrying difficulties
- To add a skeleton-noise effect
- User must be able to reset gameplay without removing cover
